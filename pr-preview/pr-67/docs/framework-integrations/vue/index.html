Vue

Vue components for the Uppy UI plugins.

Install​

  • NPM
  • Yarn
  • CDN
npm install @uppy/vue
yarn add @uppy/vue
caution

The bundle consists of most Uppy plugins, so this method is not recommended for production, as your users will have to download all plugins when you are likely using only a few.

It can be useful to speed up your development environment, so don't hesitate to use it to get you started.

<!-- 1. Add CSS to `<head>` -->
<link href="https://releases.transloadit.com/uppy/v0.0.0/uppy.min.css" rel="stylesheet">

<!-- 2. Initialize -->
<div id="uppy"></div>

<script type="module">
  import { Uppy, Box } from "https://releases.transloadit.com/uppy/v0.0.0/uppy.min.js"
  const uppy = new Uppy()
  uppy.use(Box, {
  // Options
  })
</script>
note

You also need to install the UI plugin you want to use. For instance, @uppy/dashboard.

Use​

The following plugins are available as React component wrappers:

  • <Dashboard /> renders @uppy/dashboard
  • <DragDrop /> renders @uppy/drag-drop
  • <ProgressBar /> renders @uppy/progress-bar
  • <StatusBar /> renders @uppy/status-bar

Instead of adding a UI plugin to an Uppy instance with .use(), the Uppy instance can be passed into components as an uppy prop. Due to the way Vue handles reactivity, you can initialize Uppy the same way you would with vanilla JavaScript.

<script>
    import { Dashboard } from '@uppy/vue';
    import Uppy from '@uppy/core';
    import Webcam from '@uppy/webcam';

    // Don't forget the CSS: core and UI components + plugins you are using
    import '@uppy/core/dist/style.css';
    import '@uppy/dashboard/dist/style.css';
    import '@uppy/webcam/dist/style.css';

    const uppy = new Uppy().use(Webcam);
</script>

<main><Dashboard uppy={uppy} plugins={["Webcam"]} /></main>
